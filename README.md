# Photo Selector
![App Screenshot](photo_selector.png)

Simple desktop application built with Python to browse photos quickly and mark favorites.
Selected photos are automatically copied to a chosen folder.

## Features
- View photos one by one in large size
- Navigate using buttons or keyboard arrows
- Mark photos as favorites
- Automatically copy selected photos to another folder
- Rotate images
- Display progress (current photo / total photos)

## Technologies
- Python
- Tkinter
- Pillow

## Installation
Clone the repository and install the required dependencies: 
```bash
pip install -r requirements.txt
```

## Download
Download the application from the Releases section.
Mac version: photo-selector-mac.app  
Windows version: photo-selector-windows.exe

## Run
```bash
python selector_visual.py
```
